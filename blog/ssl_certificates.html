Install SSL Certificate Using Let's Encrypt

Install certificate as seen in tutorial:

  1. Go to ACME Github repo
  2. Download ACME install script and run it. This creates acme.sh
  3. mkdir ~/pub && cd ~/pub && python3 -m http.server 80
  4. Run ./acme.sh --issue -d loic.lescoat.me -d www.loic.lescoat.me -w ~/pub/ --server letsencrypt; replace the -d arguments as required
  5. Stop HTTP server
  6. In nginx, pass acme's full-chain cert (full-chain certificate) to ssl_certificate and the cert key (private key) to ssl_certificate_key: 
    server {
    listen 0.0.0.0:443 ssl;
    listen [::]:443 ssl;

    ssl_certificate /path/to/fullchain.cer;
    ssl_certificate_key /path/to/loic.lescoat.me.key;
    ...
}

Bonus: renew SSL certificate

By default: acme auto-renews the certificate using a cron job. If this fails, renew it manually using: 

DIR=/some_path/
cd $DIR/pub
python3 -m http.server 80

# in another terminal
cd $DIR/tmp
PATH_TO_ACME_SH=$HOME/.acme.sh/.acme.sh
$PATH_TO_ACME_SH --cron --home $(basename $PATH_TO_ACME_SH)
